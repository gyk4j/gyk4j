| Dec 2023 - Mar 2024 | C#                          | :white_check_mark: | [Wreck.NET][wrecknet]: Port of [Wreck][wreck4j]. Uses [exiftool][exiftool], [7-Zip][7z] and [MediaInfo][mediainfo].                           |
